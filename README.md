# Titanic — предсказание выживаемости

Kaggle-соревнование, с которого многие начинают знакомство с ML. Решил не просто подать первый попавшийся baseline, а разобраться нормально — посмотреть на данные, подумать над признаками, подобрать модель.

**Результат: 0.77751 (Public Leaderboard)**

## Что внутри

Один ноутбук, всё по порядку:

- EDA — смотрю на пропуски, распределения, как выживаемость зависит от пола/класса/возраста
- Feature Engineering — два новых признака: размер семьи (`Col`) и флаг ребёнка (`IsChild`)
- Модель — CatBoost, выбрал потому что умеет работать с категориальными признаками и пропусками без лишнего препроцессинга
- GridSearchCV — перебор по `iterations`, `learning_rate`, `depth`
- Метрики — accuracy, ROC-AUC, confusion matrix, важность признаков

## Идеи для улучшения

Вещи, которые пока не пробовал, но скорее всего дадут прирост:

- Извлечь титул из имени (Mr, Mrs, Master...) — `Master` это почти всегда мальчик, сильный сигнал
- Флаг наличия каюты — `Cabin` пропущен у 77%, но само наличие записи коррелирует с выживаемостью
- `Fare` делить на размер группы — групповые билеты имеют один fare на всех
- Попробовать стекинг с LightGBM или логрегом сверху
